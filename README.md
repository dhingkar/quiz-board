# Quiz Board App

A visual quiz board you can project to an audience. Includes a point-and-click editor — no code editing needed.

## Setup (one time only)

1. Install Node.js from https://nodejs.org (click the LTS button)
2. Open **Terminal** (Mac) or **Command Prompt** (Windows)
3. Navigate to this folder:
   ```
   cd path/to/quiz-board-app
   ```
   (Tip: on Mac you can type `cd ` then drag the folder into Terminal)
4. Install dependencies:
   ```
   npm install
   ```
   This takes about 30 seconds. You only need to do this once.

## Run the app

Every time you want to use it, open Terminal and run:
```
cd path/to/quiz-board-app
npm run dev
```

The app opens automatically in your browser at http://localhost:5173

**To stop it:** press Ctrl+C in the Terminal.

## How to use

1. **Editor mode** opens first — set your grid size, categories, colors, and questions
2. Hit **▶ Play** to switch to the game board
3. Click **✎ Edit** in the top bar to go back and make changes
4. Your data is saved automatically — it will be there next time you open the app

## Presenting

- Connect your laptop to a projector/TV
- Press **F11** (or Cmd+Ctrl+F on Mac in Chrome) for fullscreen
- No internet connection needed — runs entirely on your computer
